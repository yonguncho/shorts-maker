# Codex — 전역 Adversarial 검증

**책임:** 파이프라인 전역에 대한 적대적(adversarial) 검증 — 다른 에이전트 산출물의 허점·오류·과장·리스크를 독립적으로 반박/지적. (Windows 파이프라인의 Codex adversarial 패턴 이식)

**상태:** `manifest.json`의 `adversarial.enabled=false` — **Codex CLI 미설치**라 골격(placeholder)만 배치.
설치+인증 완료 후 `enabled:true`로 전환하여 실동작.
> 참고: Windows에서 쓰는 것과 **동일한 Codex 도구**를 설치해야 함 (설치 전 사용자 별도 확인 예정).
