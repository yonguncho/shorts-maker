## 톤 비교 (항목별)
| 항목 | Terminal Pro | Bright News |
|---|---|---|
| 배경 | 다크 + 섹터 블러 | 노랑 + 섹터 블러(텍스처) |
| 자막 | 흰 글자 + 검정 외곽선 | 검정 글자 + 흰 외곽선 |
| 헤더 | 다크 배경, 흰 "TRADER CHO" | 노랑 배경, 검정 "TRADER CHO" |
| 뉴스카드 | 다크 카드 | 흰 카드(신문 느낌) |
| 말풍선 | 다크 + 파랑 보더 | 흰 + 파랑 보더 |
| 인상 | 블룸버그 터미널·프로 | 한국 경제TV·뉴스쇼츠 친숙 |
| 시각 점유 | 어두운 면적 많음(클린) | 밝은 면적 많음(꽉참·자극) |

## 정직성·구조 보존(불변)
- 정직성 가드(L0.6 시제·부호, L2.7 verdict), 단일 price 스냅샷, 출처 manifest, 인물 합성 금지 — **두 톤 동일 적용**.
- 씬 12개·나레이션 8씬·말풍선 3씬·마스코트 위치/모션 — 동일.

## 결정 포인트(사용자)
- Terminal Pro = 전문·신뢰·차분. 어두워 "정보 슬라이드" 인상 가능.
- Bright News = 친숙·자극·꽉참. 밝아 스크롤 정지 유리하나, 신뢰 톤은 절제 필요.
- 첨부 `contact_sheet_pair.png`(8시각 좌우) + 두 mp4로 직접 비교.

## 발견된 새 이슈
- Bright News에서 **섹터 Pexels 블러 배경이 노랑 위에 어둡게 합성**돼 약간 탁함 — Bright 톤은 배경을 흰/연노랑 단색 또는 더 낮은 alpha가 어울릴 수 있음(토큰 외 로직이라 이번 미적용).
- 면책/시점 텍스트(text_muted 회색)가 노랑 위에서 대비 약함 — Bright 채택 시 text_muted 더 진하게 권장.
